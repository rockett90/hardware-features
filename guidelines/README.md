# Guidelines

This directory contains all engineering guidelines organised by discipline.

| Section | Path |
|---|---|
| Electrical | `guidelines/electrical/` |
| PCB | `guidelines/pcb/` |
| Components | `guidelines/components/` |
| Simulation | `guidelines/simulation/` |
| Process | `guidelines/process/` |

## Adding content

Add or update a guideline via a `chore/` PR. Once a guideline is complete, add it to `.github/agents/context/company-standards.md` so it is applied automatically during AI review.
