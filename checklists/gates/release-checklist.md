# Release Checklist

> Posted automatically when the release-please Release PR opens.

## Manufacturing Pack
- [ ] Manufacturing outputs generated by CI without errors (check Actions workflow run — any ⛔ in the summary means outputs are incomplete)
- [ ] Gerbers visually verified against PCB layout
- [ ] DRC confirmed clean (zero errors)
- [ ] BOM has no TBDs — all MPNs confirmed
- [ ] CPL file verified against placement

## IVV Completion
- [ ] All TRR-gate items verified
- [ ] All P1 and P2 findings resolved or formally deferred

## Release Package
- [ ] rc tag created
- [ ] Changelog complete
- [ ] Version bump correct

## Traceability
- [ ] library.lock committed
- [ ] Sign-off docs committed
- [ ] All gate tags present

## Archive
- [ ] Release package archived

## Sign-off
- [ ] Release date recorded
- [ ] Lead name recorded
