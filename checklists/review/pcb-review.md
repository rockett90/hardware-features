# PCB Review Checklist

> Engineer reference — not auto-posted.

## Before Review
- [ ] Latest version checked out
- [ ] DRC run and results reviewed

## During Review
- [ ] Board outline correct
- [ ] Component placement logical and reviewed
- [ ] Trace widths appropriate for current
- [ ] Differential pairs routed correctly
- [ ] Ground planes poured and connected
- [ ] Silkscreen readable and not overlapping pads
- [ ] Courtyard clearances met
- [ ] Creepage and clearance checked

## After Review
- [ ] Findings raised as GitHub issues
- [ ] Review comments added to PR
